Building webforms for mod_harbour

Notas

Como desarrolladores de aplicaciones de escritorio al usar librerias GUI, solemos utilizar un editor de recursos e identificar cada control con ID's. Posteriormente 'damos vida' a esos controles definidos en recursos y los vinculamos a través del ID definido, con nuesto codigo fuente y así para dotarlos de funcionalidad. Por ejemplo, para validación, obtención de los datos, etc.

Utilizando el plugin JsonForm (o similares), el paradigma de programación descrito arriba se transforma simplemente en que nuestro script de servidor Harbour, obtiene los datos y genera unas reglas de validación para alimentar el plugin

La idea es generar un JSON en servidor, con la estructura adecuada y el plugin nos hará la 'magia' del lado del cliente (en navegador), sin tener que generar codigo HTML, ni scripts de validación.

Podemos contar con la ayuda de diseñadores visuales, tipo 'Resource workshop' en ecosistemas como formeo o form.io

Les invito a que piensen en JSON :)
